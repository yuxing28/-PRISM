## ⚠️ 测试发现的问题

### 1. 配置问题（已在测试中发现）

**问题**：
- ts-jest配置警告（使用了deprecated的globals配置）
- isolatedModules配置应该移到tsconfig.json

**影响**：
- 不影响功能，但会产生警告信息
- 未来版本可能不兼容

**建议**：
```javascript
// jest.config.js - 移除globals配置
// 改为在transform中配置

// tsconfig.json - 添加
{
  "compilerOptions": {
    "isolatedModules": true
  }
}
```

### 2. 测试覆盖不足

**缺失的测试**：
- ✅ API路由测试（`app/api/chat/route.ts`）- 已完成
- ❌ 实际的前端交互测试（需要E2E）
- ❌ 性能测试（响应时间、内存使用）
- ❌ 可访问性测试（a11y）

**影响**：
- 无法验证完整的用户流程
- 无法发现性能问题

**建议**：
- 使用Playwright添加E2E测试
- 添加性能基准测试

### 3. Mock数据与真实API可能存在差异

**问题**：
- Mock数据是手工编写的
- 没有定期与真实API对比验证
- 可能存在格式差异

**影响**：
- Mock测试通过不代表真实API一定正常
- 可能遗漏真实API的边界情况

**建议**：
- 定期运行真实API测试（每周一次）
- 记录真实API响应，更新Mock数据
- 添加Mock数据与真实API的对比测试

// __tests__/e2e/decision-flow.spec.ts
import { test, expect } from '@playwright/test';

test.describe('完整决策流程', () => {
  test('用户应该能完成一次决策分析', async ({ page }) => {
    // 1. 访问首页
    await page.goto('http://localhost:3000');
    
    // 2. 输入决策问题
    await page.fill('textarea', '我想在北京开一家咖啡店，有100万启动资金');
    
    // 3. 点击发送
    await page.click('button[type="submit"]');
    
    // 4. 等待AI响应
    await page.waitForSelector('text=/分析/', { timeout: 30000 });
    
    // 5. 验证雷达图显示
    await expect(page.locator('canvas')).toBeVisible();
    
    // 6. 验证评分显示
    const scoreText = await page.textContent('[data-testid="score"]');
    expect(scoreText).toMatch(/\d+/);
    
    // 7. 验证风险等级显示
    await expect(page.locator('text=/风险/')).toBeVisible();
  });

  test('用户应该能进行多轮对话', async ({ page }) => {
    await page.goto('http://localhost:3000');
    
    // 第一轮：信息不足
    await page.fill('textarea', '我想创业');
    await page.click('button[type="submit"]');
    await page.waitForSelector('text=/需要了解/');
    
    // 第二轮：补充信息
    await page.fill('textarea', '有50万启动资金，在成都，有3年餐饮经验');
    await page.click('button[type="submit"]');
    await page.waitForSelector('text=/根据您提供/');
    
    // 验证entropy提升（通过UI元素判断）
    const messages = await page.locator('[data-testid="message"]').count();
    expect(messages).toBeGreaterThanOrEqual(4); // 2轮对话，至少4条消息
  });

  test('用户应该能切换决策模式', async ({ page }) => {
    await page.goto('http://localhost:3000');
    
    // 切换到Fast模式
    await page.click('text=/快速/');
    
    // 输入并发送
    await page.fill('textarea', '买不买iPhone 15');
    await page.click('button[type="submit"]');
    
    // Fast模式响应应该更快更简短
    await page.waitForSelector('text=/建议/', { timeout: 15000 });
  });
});
